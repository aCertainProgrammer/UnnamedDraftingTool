Welcome to the program!

Key features:

  1. Several ways of picking champions:

    • Click a champion, then click a pick or ban slot
    • Drag champions around and drop them on picks or bans - you can also drag the picks and bans themselves
    • Hover over a champion with your mouse, then press 1-10 (10 is the 0 key) to pick the hovered champion
    • Press 1-10 (10 is the 0 key) to pick the first champion in the champions list
    • Whether you pick or ban a champion with the shortcut is dependent on your current mode! Toggle it by clicking the button or by using a shortcut
  2. Many keyboard shortcuts, check out the manual in the top left corner of the program to learn about them
  3. Quick and easy searching with automatic search bar refocusing and other QoL improvements
  4. Multiple color schemes, toggle them with the button in the top-left corner
  5. The "All", "Ally" and "Enemy" icons are simply different slots for you to store champion pool data. Learn how to input the data using the manual, then click the input button in the top right corner
  6. Your picks and bans are saved, no worries about losing them on page reload
  7. You can clear the draft quickly using Delete

This colors the borders of the champions depending on which team they belong to:

- green for ally champions

- red for enemy champions

- orange for champions in both teams

- champions that are not in either ally or enemy teams are not given a colored border

This option decides which data source the program loads on startup (or when the page is reloaded).

If enabled, your custom data will be loaded, provided it is available.

If enabled clears the search bar whenever it is refocused by pressing spacebar or typing a letter

Legacy search means pure substring matching - if a word contains the exact string of the search query, it will be matched - for example, "ji" will match "JInx"

Modern search means that if a word contains the letters from the search query in the same order, it will be matched - for example, "ji" will match "JInx", "JhIn" and "seJuanI".
